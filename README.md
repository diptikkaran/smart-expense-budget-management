# Smart Expense & Budget Management System

## Overview

This project was developed as part of my B.Tech learning to help users manage their daily income, expenses, and student group expenses in a simple way. It uses browser Local Storage to save data and generate financial reports.

## Features

- User Registration & Login
- Forgot Password
- Dashboard
- Personal Income & Expense Tracking
- Group Expense Management
- Reports
- Savings Rate Calculation
- Recent Transactions

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage
- Git & GitHub
